# 🖼️ 图片编辑工具

基于 **Node.js + Vue3 + Sharp** 的本地图片编辑工具，支持文件夹浏览、图片裁剪、旋转、放大缩小、图片压缩和批量处理。

## 功能特性

✅ **文件夹浏览** - 使用浏览器原生 `showDirectoryPicker` API 选择本地文件夹，列出所有图片
✅ **图片编辑** - 裁剪 (cropperjs)、旋转、缩放查看
✅ **单张压缩** - 支持 JPEG/PNG/WebP/AVIF 格式转换，可自定义压缩质量
✅ **批量压缩** - 批量修改分辨率、转换格式，一次性下载所有结果
✅ **即时下载** - 编辑结果直接下载到本地，不会上传到第三方服务器

## 项目结构

```
imageTool/

```

## 快速开始

### 安装依赖

```bash
# 安装后端依赖
cd server
npm install

# 安装前端依赖
cd ../client
npm install
```

### 启动服务

```bash
# 在一个终端启动后端 (服务器 3000 端口)
cd server
npm start

# 在另一个终端启动前端 (开发服务器 5173 端口)
cd client
npm run dev
```

然后打开浏览器访问 `http://localhost:5173`

## 使用说明

### 1. 选择文件夹

点击右上角 **📁 选择文件夹**，选择你的图片所在文件夹，左侧会列出所有图片。

### 2. 图片编辑

点击左侧图片开始编辑：
- ✂️ 裁剪: 点击裁剪按钮，拖动选择区域，确认后裁剪
- ↺/↻: 左右旋转 90 度
- 缩放滑块: 缩放图片显示大小
- 🗜️ 压缩: 选择质量和目标格式，点击压缩

### 3. 批量压缩

点击 **📦 批量处理** 打开面板：
- 设置压缩质量、目标格式
- 可限制最大宽度/高度（保持比例缩放）
- 点击批量处理，会自动下载所有处理后的图片到浏览器下载目录

## 技术栈

- **后端**: Node.js, Express, Sharp (高性能图片处理)
- **前端**: Vue 3, Vite, Cropper.js, Axios
- **裁剪**: cropperjs (成熟专业的前端裁剪库)

## 注意事项

- 需要 Chrome/Edge 或支持 File System Access API 的现代浏览器
- 所有处理均在本地完成，不会上传到云端
- 批量处理直接下载到浏览器下载文件夹
